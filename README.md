# Racing Tracker

A race-style React web app for tracking 2026 motorsport weekends in one clean pile.

## Series included

- Formula 1
- FIA World Endurance Championship
- DTM
- GT World Challenge Europe
- British GT Championship
- Major standalone races

## How it works

- The **All upcoming** tab only shows races that have not finished yet.
- Each series tab shows the full 2026 season.
- Finished races stay in their series tab but are greyed out.
- The app includes track names, locations, countries, dates, and race formats.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
